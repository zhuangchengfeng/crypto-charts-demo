# CryptoChart - 加密货币回测图表系统

基于 TradingView Lightweight Charts 的币安合约 K 线回测工具，支持技术指标、画线工具、时光机复盘和模拟交易。

## 功能

- **K 线图表** — 支持 1/2 图模式，7 个周期（15m / 1h / 4h / 1d / 3d / 1w / 1M）
- **技术指标** — SMA、EMA、布林带、RSI、MACD、FVG、Order Blocks、支撑阻力
- **画线工具** — 趋势线、水平线，拖拽编辑，右键删除
- **时光机** — 选择亏损仓位记录，逐根回放历史行情，支持键盘 P 键播放
- **模拟交易** — 多仓位管理，止损止盈，浮动盈亏
- **多周期联动** — MTF 高低点标记
- **扫描面板** — `/t` 扫描器，支撑阻力位计算

- 本项目为DEMO阉割版本,于完整版本使用的工具不一样
- 下载后在文件夹里双击start.bat等待启动网页即可,默认端口5001
